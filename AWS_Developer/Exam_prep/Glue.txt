AWS Glue :
	 is a serverless data integration service that makes it easy to discover, prepare, and combine data for analytics, machine learning, and application development.
	AWS Glue provides all of the capabilities needed for data integration so that you can start analyzing your data and putting it to use in minutes instead of months. 
	Glue is not best suited to handle real-time data.
	
	AWS Glue is a fully managed extract, transform, and load (ETL) service that makes it easy for customers to prepare and load their data for analytics.
